# Employee Attendance Management System

## Project Overview
The Employee Attendance Management System is a desktop application developed using **C# and .NET Framework Windows Forms**. The purpose of this system is to manage employee records and maintain daily attendance in a simple and user-friendly way.

## Features
- User Login Authentication
- Dashboard Management
- Add New Employees
- Update Employee Information
- Delete Employee Records
- Search Employees
- Mark Employee Attendance
- Update and Delete Attendance Records
- Attendance Reports
- Database Connectivity using ADO.NET and SQL Server

## Technologies Used
- C# Programming Language
- .NET Framework Windows Forms
- Microsoft Visual Studio 2022
- SQL Server / LocalDB
- ADO.NET
- Git & GitHub

## Installation and Running Steps

1. Clone or download this repository.
2. Open the project in Microsoft Visual Studio.
3. Restore required packages (if any).
4. Configure the SQL Server database connection in `App.config`.
5. Build the solution.
6. Press `F5` to run the application.
